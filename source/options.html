The format above is used to render the commit message displayed in the prompt after searching for a bug.

There are a bunch of variables which expand in the format. To place a variable named title in the format, use %(title)s, much like Python's dict string formatting specifiers.

(This is because the string formatting function was written by a Python enthusiast, and I'm one, too)

Variable Description Example Value
%(id)s The ID of the story 29784057
%(title)s The title of the story Profile edit page should allow editing of name
%(titlefull)s The title of the story, including the owner's initials, if there is an owner. Profile edit page should allow editing of name (ZK)
%(owner)s The full name of the story owner, if there is one. Zach Kanzler
%(initials)s The initials of the story owner, if there is one. ZK
%(labels)s All the labels applied to the story, separated by spaces only. mylabel altbranch testing
%(labels_comma)s All the labels applied to the story, separated by commas only. mylabel,altbranch,testing
%(labels_comma_space)s All the labels applied to the story, separated by commas and spaces. mylabel, altbranch, testing
